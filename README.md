## ğŸ”® About The Project

This project focuses on **data preparation and exploratory data analysis** for a public health use case.

The objective is to transform raw, heterogeneous food product data into a **clean, analyzable dataset** that can support:
- Public health studies
- Nutritional analysis
- Decision-making and reporting

Key aspects of the project include:
- Data cleaning and validation
- Handling missing and inconsistent values
- Univariate and multivariate analysis
- Generation of exploratory visual insights

### ğŸ“Š Dataset

The dataset used in this project comes from **Open Food Facts**, an open and collaborative database of food products from around the world.

Data source:
- https://world.openfoodfacts.org/

The dataset contains information such as:
- Product categories
- Nutritional values
- Ingredients
- Labels and tags
- Country-specific metadata

## ğŸª„ Getting Started

This project is delivered as a **Jupyter Notebook** and can be run locally in a Python environment.

### ğŸ“˜ Prerequisites

- Python 3.x
- Jupyter Notebook or JupyterLab

### ğŸ§™ Installation

1. Clone the repository:

       git clone https://github.com/joelle-jnbaptiste/SchoolProject---Public-health-data-preparation.git

2. Create and activate a virtual environment (optional but recommended).

3. Install dependencies:

       pip install pandas numpy matplotlib seaborn jupyter

4. Launch Jupyter:

       jupyter notebook

5. Open the notebook and run the cells sequentially.
